# portfolio

Portfolio site built with **Vite**, **React**, **TypeScript**, and **Tailwind CSS v4**.

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

## Deploy on Vercel

Connect this repository in the Vercel dashboard. Vercel detects **Vite** and uses `npm run build` with output directory **`dist`** by default.

## Images

The design references PNGs under `src/assets/` (Figma export hash filenames). The repo includes tiny placeholder images so the app builds; replace them with your real exported assets to restore the full layout.
